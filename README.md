**Metalearner-Based Average Treatment Effect Estimation: MLR vs. UMLR**
> Supplementary code for the manuscript *Trustworthy AI/ML Regression and Unbiased Causal Inference for Real-World Data* (available on [arXiv:2605.24377](https://arxiv.org/abs/2605.24377)).  
> This repository illustrates how the proposed **Unbiased Machine Learning Regression (UMLR)** can be incorporated into metalearners to estimate average treatment effects (ATEs) from observational data, and effectively reduce systematic prediction bias.
